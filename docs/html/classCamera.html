Detailed Description

3D camera controller for orbital visualization

Implements a spherical coordinate camera system that orbits around a central point (typically the central body). Features include:

  • Spherical coordinate navigation (azimuth, elevation, distance)
  • Smooth zoom controls with distance constraints
  • Mouse-based rotation controls
  • View and projection matrix generation
  • Debug coordinate axis rendering
  • Position display utilities

The camera is designed specifically for orbital mechanics visualization where the primary interest is viewing orbits and transfers around a central gravitational body.
