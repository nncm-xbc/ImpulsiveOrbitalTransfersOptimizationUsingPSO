Detailed Description

Solves Lambert's problem for orbital trajectory determination.

Lambert's problem: Given two position vectors and a time of flight, find the orbit that connects them. This is a fundamental problem in orbital mechanics used for:

  • Interplanetary trajectory design
  • Orbital rendezvous planning
  • Transfer orbit optimization

This implementation uses a robust numerical approach with universal variables and Stumpff functions for improved convergence.
