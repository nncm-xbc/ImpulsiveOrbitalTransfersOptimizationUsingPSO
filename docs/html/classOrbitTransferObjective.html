Detailed Description

template<typename T, typename Fun>
class OrbitTransferObjective< T, Fun >

Objective function for orbital transfer optimization using PSO.

Template Parameters
TFloating point type (typically double)
FunFunction type for the objective function

This class encapsulates the orbital transfer optimization problem as an objective function that can be minimized by PSO. It handles:

  • Coplanar circular-to-circular transfers
  • Non-coplanar circular-to-circular transfers
  • Elliptical orbit transfers
  • Constraint handling for feasible trajectories

The optimization variables represent:

  • x[0]: Departure true anomaly (0 to 2π)
  • x[1]: Arrival true anomaly (0 to 2π)
  • x[2]: First impulse magnitude
  • x[3]: Second impulse magnitude
  • x[4]: First impulse direction
  • x[5]: Time of flight
