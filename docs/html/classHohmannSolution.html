Detailed Description

template<typename T>
class HohmannSolution< T >

Calculates the exact Hohmann transfer solution between two circular coplanar orbits.

Template Parameters
TFloating point type (float or double)

The Hohmann transfer is the minimum energy two-impulse transfer between two circular coplanar orbits. This class provides the analytical solution for validation purposes.

The Hohmann transfer is globally optimal only when the radius ratio is within specific bounds (approximately 0.08376 ≤ R₂/R₁ ≤ 15.582). Outside these bounds, a three-impulse bi-elliptic transfer may be more efficient.

Compute the Hohmann transfer solution.

Returns
Complete solution structure with ΔV components

Calculates the analytical Hohmann transfer using:

  • ΔV₁ = √(μ(2/r₁ - 1/a)) - √(μ/r₁)
  • ΔV₂ = √(μ/r₂) - √(μ(2/r₂ - 1/a)) where a = (r₁ + r₂)/2 is the transfer orbit semi-major axis
