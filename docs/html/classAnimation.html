Detailed Description

Controls animation timing and playback for orbital visualizations.

Manages animation state and timing for orbital transfer visualizations. Features include:

  • Play/pause control
  • Variable speed adjustment
  • Looping support
  • Progress tracking (0.0 to 1.0)
  • Real-time updates based on frame delta time

The animation system is designed to smoothly animate orbital transfers, allowing users to visualize spacecraft motion along transfer trajectories at controllable speeds.
